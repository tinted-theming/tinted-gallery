# Tinted Gallery

https://tinted-theming.github.io/tinted-gallery

This gallery is a collection of tinted themes maintained by [tinted-theming].
The software license of `build.sh` is CC0.

## Contribution

For easier dependency management, install [Nix] (including [Nix Flakes]) and
run `nix develop` to enter a shell with all dependencies available for
development and debugging.

[tinted-theming]: https://github.com/tinted-theming
[Nix]: https://nixos.org/
[Nix Flakes]: https://wiki.nixos.org/wiki/Flakes
